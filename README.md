## Project info

AI-powered personal trainer that uses MediaPipe pose detection to analyze bicep curl form in real time. Tracks joint angles, detects form violations (elbow drift, flare, shoulder shrug), and auto-validates reps based on biomechanical thresholds. Built with React, TypeScript, and Tailwind CSS.

More exercises will be added soon!

**URL**: https://pose-coach-go.lovable.app

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
